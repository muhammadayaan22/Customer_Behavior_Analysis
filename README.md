## Overview
This project demonstrates an end-to-end Data Analytics workflow, starting from raw data to actionable business insights.  
The dataset is processed and analyzed using Python (Jupyter Notebook), stored and queried in PostgreSQL using SQL, and visualized through an interactive Power BI dashboard. Final insights are summarized in a report and presentation.

This project highlights practical skills in data cleaning, analysis, database querying, and data storytelling.

---

## Dataset
- Structured business/transactional dataset
- Contains multiple attributes such as customers, categories, and orders
- Used for trend analysis, aggregations, and performance insights
- Format: CSV/Excel

---

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- PostgreSQL
- SQL
- Power BI
- Gamma (Presentation)

---

## Project Workflow
1. Load dataset in Jupyter Notebook using Python
2. Perform data cleaning and preprocessing
3. Conduct Exploratory Data Analysis (EDA)
4. Export cleaned data to PostgreSQL
5. Write SQL queries for aggregations and insights
6. Build interactive dashboard in Power BI
7. Create report and presentation

---

## Results
- Cleaned and structured raw data for analysis
- Identified trends, patterns, and KPIs
- Extracted top-performing categories/items using SQL
- Built an interactive Power BI dashboard for visualization
- Delivered insights through reports and slides
